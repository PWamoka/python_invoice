# Python Flask Invoice — Toolkit Document

## 1) Title & Objective
**Tech:** Python + Flask  
**Objective:** Build a simple invoice system that uses Python and Flask.

**Objective:** Build a minimal REST API for notes to learn Rust’s backend ecosystem and document learning with GenAI.

## 2) Quick Summary of the Technology
**Python:** A versatile, high-level programming language known for readability and simplicity. .  
**Flask:** A lightweight Python web framework used to build web applications.  
**Real-world usage:** Small Businesses & Startup: Freelancers, consultants, or small shops can generate invoices for clients without expensive software..

## 3) System Requirements
- OS: Linux (Ubuntu/CentOS), macOS, or Windows 10/11. Production: a Linux server is recommended.
- Python: 3.10+ (3.11 recommended). Use the official CPython builds.
- Package manager: pip (or Poetry/Poetry2/Poetry v1+ if you prefer). Use virtual environments (python -m venv .venv).
- Web Browser e.g. Google chrome, Mozilla Firefox, Edge Safary; to view and interact with the rendered invoice application.

## 4) Installation & Setup
```
1. Download and install Python 3.9+ from python.org (Verify installation: (python –version)
2. Create Project Folder (mkdir invoice-toolkit, cd invoice-toolkit)
3. Create Virtual Environment (Recommended)
4. Install Required Packages (Flask, pip install pandas openpyxl openai python-dotenv)
5. Set up Project structure
	invoice-toolkit
		.env
		docs
		templates
6. Run the Flask App (Open browser and go to: http://127.0.0.1:5000/)
